# AI IDE工具使用统计分析工具

## 功能介绍

这是一个用于分析AI编程IDE工具使用日志的Python应用程序，主要功能包括：

1. **文件选择**：可以选择要分析的log文件（支持1_model_log.md、2_model_log.md等）
2. **工具信息显示**：以表格形式显示每个工具的调用信息，包括序号、思考、工具名称、状态和其它信息
3. **详细信息查看**：点击表格中的条目，可以查看该条目的完整信息
4. **统计分析**：显示工具使用的统计信息，包括使用次数、成功/失败/运行中状态统计

## 使用方法

1. **启动工具**：运行`python tool_analyzer.py`
2. **选择文件**：在文件选择下拉框中选择要分析的log文件，或点击"选择文件"按钮手动选择
3. **查看工具信息**：表格中会显示所有工具调用的信息
4. **查看详细信息**：点击表格中的任意条目，下方会显示该条目的完整信息
5. **查看统计信息**：统计信息区域会显示工具使用的统计数据

## 技术实现

- 使用Python的tkinter和ttk库创建GUI界面
- 使用正则表达式解析log文件内容
- 采用上下布局，分为四个主要部分：文件选择、工具信息表格、详细信息查看和统计信息

## 文件格式要求

log文件应包含以下格式的内容：
```
思考内容...

toolName: 工具名称
status: 状态
其它信息...
```

## 注意事项

- 工具会自动识别当前目录下的所有_model_log.md文件
- 思考内容来源于上一个分段的最后一句非空句子
- 工具名称和状态是固定的信息
- 其它信息包括工具调用的参数和返回结果
