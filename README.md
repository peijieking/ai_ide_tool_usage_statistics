使用python ttk 写一个统计 AI 编程IDE 工具使用统计的工具分析当前目录下的log文件
# 主要需求
- 界面采用上下布局
- 第一部分为log选择,以便我选择要分析的文件
- 第二部分为treeview,用于显示每一个工具的信息,列名是 [序号] [思考] [工具名称] [状态] [其它信息]
- 第三部分为 点击 treeview 条目后,该条信息的完整显示
- 第四部分为文件里工具的使用统计,包括使用了哪些工具,每个工具使用了多少次,成功,失败统计
注意：对log使用toolName 进行分段，但 [思考]列来源于上一个分段最后一句非空句子， [工具名称] [状态]是固定信息，[其它信息]为本段剩余信息
