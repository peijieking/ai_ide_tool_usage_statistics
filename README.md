# AI IDE 工具使用统计分析器

一个基于 Python Tkinter 的 GUI 工具，用于分析 AI 编程 IDE 的工具使用日志文件。

## 功能特性

- 📁 **文件选择**：支持选择多个日志文件进行分析
- 📊 **工具使用记录**：以表格形式展示每个工具的详细使用信息
- 📋 **详细信息**：点击工具记录查看完整信息
- 📈 **统计信息**：统计每个工具的使用次数、成功次数和失败次数

## 界面布局

应用程序采用上下布局，分为四个主要部分：

1. **日志文件选择**：用于选择要分析的日志文件
2. **工具使用记录**：显示每个工具的使用记录，包括序号、思考、工具名称、状态和其它信息
3. **详细信息**：显示选中工具记录的完整信息
4. **工具使用统计**：统计所有工具的使用情况

## 使用方法

### 环境要求

- Python 3.x
- tkinter（通常随 Python 安装）

### 运行程序

```bash
python tool_usage_analyzer.py
```

### 分析日志文件

1. 程序启动后会自动加载当前目录下的所有 `_model_log.md` 文件
2. 从下拉菜单中选择要分析的日志文件
3. 或点击「浏览」按钮选择其他日志文件
4. 工具使用记录和统计信息会自动更新
5. 点击工具使用记录表格中的任意条目，可在下方查看详细信息

## 日志文件格式

日志文件应符合以下格式：

```
[思考内容]
[工具名称] [状态] [其它信息]
[工具名称] [状态] [其它信息]
...
```

其中：
- **思考内容**：来自上一个分段的最后一句非空句子
- **工具名称**：使用的工具名称
- **状态**：工具执行状态（如：完成、成功、失败等）
- **其它信息**：工具使用的详细信息
