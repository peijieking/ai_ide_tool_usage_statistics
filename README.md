# AI IDE 工具使用统计分析器

一个基于 Python Tkinter/ttk 的图形界面工具，用于分析 AI 编程 IDE 生成的工具使用日志文件。

## 功能特性

1. **文件选择**：支持选择多个 log 文件进行分析
2. **工具使用记录**：以表格形式显示所有工具调用记录
3. **详细信息**：点击表格条目可查看完整的工具调用信息
4. **使用统计**：统计每种工具的使用次数、成功次数和失败次数

## 使用方法

### 1. 安装依赖

本工具使用 Python 标准库，无需额外安装依赖：
- Python 3.x
- tkinter (通常随 Python 一起安装)
- ttk (tkinter 的主题化部件库)

### 2. 运行程序

```bash
python tool_usage_analyzer.py
```

### 3. 操作步骤

1. 启动程序后，会自动加载当前目录下的 log 文件
2. 使用下拉菜单选择要分析的 log 文件
3. 点击 "分析" 按钮开始分析
4. 在表格中查看工具使用记录
5. 点击任意条目查看详细信息
6. 在统计信息区域查看工具使用统计

## 文件格式

工具支持分析 Markdown 格式的 log 文件，文件名通常为：
- 1_model_log.md
- 2_model_log.md

日志文件应包含 `toolName`、`status` 等关键字段用于分析。

## 界面说明

### 第一部分：Log 文件选择
- 下拉菜单：显示可用的 log 文件
- 浏览按钮：手动选择 log 文件
- 分析按钮：开始分析选中的文件

### 第二部分：工具使用记录
- 序号：工具调用的序号
- 思考：上一次工具调用后的思考内容
- 工具名称：使用的工具名称
- 状态：工具调用的状态（成功/失败）
- 其它信息：工具调用的完整信息

### 第三部分：详细信息
- 显示选中工具调用的完整信息

### 第四部分：使用统计
- 每种工具的使用次数
- 成功次数和失败次数
- 总计工具种类和使用次数
